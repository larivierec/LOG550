Description

Le but du systeme sur le microcontroleur est de convertir le signal analogique en donnees numeriques avec l'ADC et de les transferer vers le PC par lien en serie soit en:

Le but du deuxieme laboratoire de LOG550 est identique a la premiere.
Nous n'utilisons pas directement l'ASF (Atmel Software Framework), nous utilisons plutot un noyau uCOS-II qui interface avec le EVK1100

  • Vitesse de transfert utilisée
    • 115200 bauds (Bonus - USART0)
    • 57600 bauds (ADC - USART1)
  • Logiciel necessaire : AVR-Studio 6.2, AVR-GCC, Oscilloscope JAVA, TeraTerminal PRO pour Bonus
  • Materiel necessaire : ATmega32 on EVK1100 board
  • Librairies/Modules necessaire :
    • uCOS-II de Microcium qui est inclut dans le fichier zip.
  • Specifications
    • 2 canaux operant 500 echantillons/sec
    • Tout acces a une ressource partagee requiert un semaphore
    • L'utilisation d'un FIFO fournis par le noyau uCOS-II
    • 5 taches dont: LED_Flash, UART_Cmd_RX, UART_SendSample, ADC_Cmd(), Alarm_MsgQ
    • 1 tache (boni): Bonus_Thread() qui s'occupe d'afficher en terminal de l'utilisation CPU et Le nombre de context switch
